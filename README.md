# Nivasyaa - End to End Interior Design Solutions

A modern, responsive website for **Nivasyaa**, an end-to-end interior design services company specializing in residential interiors.

## 🚀 Quick Start

```bash
npm install
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000)

## 📁 Project Structure

- `src/app/` - Next.js pages (home, portfolio, services, about, blog, contact, booking)
- `src/components/` - Reusable React components
- `tailwind.config.js` - Tailwind CSS configuration with brand colors
- `next.config.js` - Next.js configuration
